# ADR 0001

## Contexto

O projeto possui poucas funcionalidades, porém será desenvolvido utilizando arquitetura semelhante à encontrada em sistemas reais.

## Decisão

Foi adotada uma variação da Clean Architecture.

## Motivos

- separar responsabilidades
- facilitar testes
- permitir troca de banco
- reduzir acoplamento
- aumentar legibilidade
- facilitar manutenção

## Consequências

Mais arquivos.

Maior organização.

Maior facilidade para evolução futura.

Menor custo de manutenção.
